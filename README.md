##Setup


#### download the repository  

```bash
git clone https://github.com/Ankits29/yachtly-full-stack-dev-test-testdemo.git
```

#### run the following command

```bash
npm install
```
#### go to front-end folder  

```bash
cd front-end
```
#### install front end dependencies  

```bash
bower install
```
#### move to root folder  

```bash
cd .. 
```
#### run

```bash
node app.js
```
